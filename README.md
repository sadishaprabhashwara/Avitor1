# 🚀 AstroCrash – Multiplayer Crash Game

## Quick Start on Replit

1. Import this project to Replit
2. In Replit Shell, run: `npm install`
3. Set environment variables in Replit Secrets:
   - `ADMIN_PASSWORD` — your admin password (default: admin123)
   - `SESSION_SECRET` — a long random string
4. Click **Run** — the game starts on port 3000

## Files
- `server.js` — Full Node.js backend (Express + Socket.io)
- `public/index.html` — Complete frontend UI
- `package.json` — Dependencies
- `.env.example` — Environment variable reference

## Admin Panel
Press **Shift+A** in the browser, enter your admin password to access:
- Live player monitoring
- Force crash / instant crash
- Speed & probability sliders
- Balance adjustments
- Bot controller
- Withdrawal & deposit management
- CSV export
- Audit logs
- And much more!
